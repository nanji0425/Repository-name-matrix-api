FAQ

常见问题。

把用户上手前最容易问的问题放在底部，避免首页变成说明书。

API 地址应该填哪个？

默认建议使用 https://matrixapi.online 主站点；也可以使用 https://matrixapi.online/v1 作为 OpenAI 兼容入口。

两个模型怎么计费？

GPT-5.6 采用官方 1:1 计费标准，Claude 4.8 Fable 5 采用官方计费标准的 2× 倍率计费。

购买套餐后怎么兑换？

在发卡站购买额度卡，拿到卡密后进入控制台钱包管理，选择额度充值里的兑换码充值，额度会实时到账。

支持哪些 SDK 和工具？

OpenAI 兼容 SDK、Anthropic SDK、Cursor、Cline、Continue、Aider、LangChain 都可接入。
