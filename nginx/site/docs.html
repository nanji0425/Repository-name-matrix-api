MatrixAPI integration docs

Use MatrixAPI as a single OpenAI-compatible endpoint for chat, model listing, image generation, token management, usage logs, and balance top-up. These docs describe MatrixAPI only.

Create token Quick start Base URL: https://matrixapi.online/v1

01

Quick start

1. Create an account

Open the dashboard and sign in or create a MatrixAPI account.

2. Create a token

Go to Tokens, create an API token, then copy it once and store it securely.

3. Call the API

Use the base URL below with OpenAI-compatible SDKs and tools.

02

Endpoints

Use the same host for all OpenAI-compatible requests.

https://matrixapi.online/v1 
POST https://matrixapi.online/v1/chat/completions
Authorization: Bearer YOUR_TOKEN_HERE

{
  "model": "gpt-5.4",
  "messages": [
    { "role": "user", "content": "Hello MatrixAPI" }
  ]
}

03

Client import

MatrixAPI tokens work with OpenAI-compatible clients. Use the dashboard token import panel for supported tools.

CC Switch

Import MatrixAPI as an OpenAI-compatible provider.

Cherry Studio

Use the same API key and base URL from MatrixAPI.

OpenAI SDKs

Set the SDK base URL to the MatrixAPI /v1 endpoint.

04

Billing and top-up

Online top-up is Alipay only. MatrixAPI uses balance-based billing and displays model costs in the dashboard pricing pages. A top-up quantity of 100 corresponds to an actual payment of 10 CNY.

Open wallet

05

Troubleshooting

If a request fails, check that your token is enabled, your selected model is available to the token group, and your client base URL ends with /v1. Contact support without sharing full tokens.
